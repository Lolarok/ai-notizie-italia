# 🤖 AI Notizie Italia - WordPress Theme

**Portale italiano di notizie sull'Intelligenza Artificiale**

## Panoramica
Tema WordPress dark mode professionale per un portale notizie AI in italiano.
Costruito con design moderno, ottimizzato per SEO e velocità.

## Struttura
```
ai-notizie-italia/
├── wordpress/
│   └── themes/
│       └── ai-italia-theme/    ← Tema principale
│           ├── style.css       ← Stili + header tema WP
│           ├── functions.php   ← Funzioni PHP
│           ├── header.php      ← Template header
│           ├── footer.php      ← Template footer
│           ├── index.php       ← Homepage
│           └── js/
│               └── main.js     ← JavaScript
├── docs/
│   └── SETUP_GUIDE.md          ← Guida installazione
└── tools/
    └── rss_sources.txt         ← Fonti RSS italiane AI
```

## Installazione Rapida (Gratis)
1. Vai su [InstaWP.com](https://instawp.com) → crea sito WordPress gratis
2. Dashboard WP → Aspetto → Temi → Aggiungi nuovo → Carica tema
3. Comprimi la cartella `ai-italia-theme` in ZIP e carica
4. Attiva il tema
5. Installa i plugin consigliati (vedi SETUP_GUIDE.md)

## Fonti RSS Integrate
- ainews.it, ai4business.it, iaol.it (Italiane)
- TechCrunch AI, VentureBeat AI (Internazionali)
- OpenAI Blog, DeepMind Blog, HuggingFace Blog
- ArXiv cs.AI (Paper accademici)

## Plugin Raccomandati (tutti gratuiti)
- **Feedzy RSS Feeds** → importa articoli automaticamente
- **Yoast SEO** → ottimizzazione motori di ricerca
- **WP Rocket** (alternativa gratis: W3 Total Cache) → velocità
- **MailPoet** → newsletter integrata
- **WP GDPR Compliance** → cookie banner GDPR
